NOISE DETECTION SYSTEM

This demo is a prototype of a sound analysis and noise detection system. The system utilizes machine learning techniques in analysising an audio sound (in .mp3 or .wav format), and produce noise density score by which the the audio is classified as being "Noisy" or "Not Noisy".

How to Use the System

1. Select an Audio File. The audio file should be in .mp3 or .wav format and the file size should be greater 100KB and lesser than 25MB. For quality result, the audio file should be within the range of 1 minutes and 5 minutes.

2. Click on the analyse button and wait for the result to be displayed. You can use the Graphical Visualisation of the audio file to gain more insights. The system provides a graphical visualisation of the audio in Wave Form, Frequency Spectrum, Spectrogam and Mel Spectrogam.
